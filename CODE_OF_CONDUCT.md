# Code of Conduct

The K2s project is committed to providing a welcoming, respectful, and harassment-free environment for everyone.

We expect all participants to act professionally and respectfully in all project spaces, including GitHub issues, pull requests, discussions, meetings, documentation, and community channels.

## Expected Behavior

- Be respectful and considerate in technical discussions.
- Welcome contributors with different levels of experience and different backgrounds.
- Focus feedback on the work and its impact on the project.
- Accept responsibility for mistakes and work constructively toward resolution.
- Respect project maintainers' decisions about scope, priorities, security handling, and release readiness.

## Unacceptable Behavior

Unacceptable behavior includes harassment, intimidation, discrimination, personal attacks, sustained disruption, or any other conduct that undermines a respectful and productive project environment.

## Reporting

If you experience or witness unacceptable behavior, please report it to:

```text
dieter.krotz@siemens-healthineers.com
```

Reports will be handled with care and confidentiality by the project maintainers.
